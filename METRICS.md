# Metrics Framework

## North Star Metric
Actioned Insights per Month

## Input Metrics
- Feedback processed per day
- Cluster accuracy (manual validation)
- Time to insight

## Outcome Metrics
- Reduction in recurring issues
- Product cycle time improvement

## Guardrail Metrics
- False-positive insight rate
- Model confidence distribution

## Review Cadence
- Weekly PM review
- Monthly leadership review
